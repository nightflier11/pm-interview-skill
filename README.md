# PM Interview Skill for Claude Code

A structured mock interview coaching skill for product management roles — from APM to CPO. It researches your target company, generates role-calibrated questions, runs mock Q&A, and coaches you toward interview-ready answers.

---

## What it does

1. **Orient** — Establishes your role level, target company, focus areas, and preferred session flow
2. **Company Research** — Builds a Strategic Context Brief using live web research: strategy, competitive landscape, product specifics, and leadership signals
3. **Question Generation** — Creates 8–12 company-specific questions calibrated to your level and focus areas
4. **Mock Q&A Coaching** — Runs one question at a time with structured feedback, targeted coaching prompts, and a per-answer scorecard
5. **Output (optional)** — Exports a prep brief (`.docx`) or answer bank (`.md`) with polished answers and notes

---

## Role coverage

| Level | Question focus |
|---|---|
| APM | Frameworks, user empathy, basic prioritisation, learning agility |
| PM | Execution, metrics, cross-functional collaboration, delivery under ambiguity |
| Senior PM | Product strategy, roadmap ownership, stakeholder influence, trade-off reasoning |
| Group PM / Director | Multi-team leadership, org design, platform thinking, build vs. buy, M&A |
| VP / CPO | Vision, company-level strategy, board-level communication, culture, talent |

---

## How to use

1. Install [Claude Code](https://claude.ai/code)
2. Add this skill to your Claude Code skills directory
3. Start a session — Claude will ask for your role, company, focus areas, and flow preference
4. Work through the coaching phases at your own pace

---

## Scoring dimensions

Each mock answer is scored across 5–7 dimensions specific to the question, for example:

- Strategic clarity
- Use of concrete examples
- Metrics grounding
- Trade-off reasoning
- Stakeholder awareness
- Failure / learning signal

Ratings: **Excellent · Strong · Needs sharpening · Missing**
